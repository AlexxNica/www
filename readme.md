# www
> The Haumea website sourcecode

### install
```sh
$ yarn
# or use npm
$ npm install
```

### build
```sh
$ yarn global add gulp-cli
# or use npm
$ npm install -g gulp-cli

$ gulp build
$ gulp watch
```

### run
```sh
$ node server.js
```
